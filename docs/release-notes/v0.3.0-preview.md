# GlucoDesk v0.3.0-preview

GlucoDesk v0.3.0-preview is a preview release focused on turning GlucoDesk into a complete desktop glucose awareness companion.

This release includes the polished dashboard experience, local-first history workflows, privacy mode, glucose awareness notifications, menu bar and tray companion flows, glycemic diary export, release-readiness checks, and installable macOS and Windows preview packages.

## Supported platform

| Platform            | Status            | Package type                                       |
| ------------------- | ----------------- | -------------------------------------------------- |
| macOS Apple Silicon | Preview supported | macOS arm64 installable ZIP containing DMG         |
| macOS Intel         | Preview supported | macOS x64 installable ZIP containing DMG           |
| Windows x64         | Preview supported | Windows x64 installable ZIP with setup installer   |
| Linux               | Not supported yet | Not available                                      |

macOS packages are not notarized yet.

Windows packages are not code-signed yet.

First-launch approval may be required on both platforms.

## Installation

Download one of the installable ZIP packages from the GitHub Release assets.

Do not use the green GitHub Code button to install GlucoDesk. That downloads the source code, not the ready-to-run app.

Available packages:

- GlucoDesk-0.3.0-preview-macos-arm64-installable.zip
- GlucoDesk-0.3.0-preview-macos-x64-installable.zip
- GlucoDesk-0.3.0-preview-windows-x64-installable.zip

For macOS:

1. Download the package matching your Mac architecture.
2. Extract the ZIP.
3. Open the DMG.
4. Drag GlucoDesk.app to Applications.
5. If macOS blocks the app, approve it from System Settings > Privacy & Security.

For Windows:

1. Download the Windows x64 package.
2. Extract the ZIP.
3. Run the setup executable.
4. Follow the installation wizard.
5. Launch GlucoDesk from the Start Menu.

## Known limitations

- GlucoDesk is not a medical device.
- GlucoDesk must not be used for treatment decisions.
- Glucose awareness notifications are not medical alarms.
- Native notifications can be delayed, blocked or hidden by the operating system.
- macOS packages are not signed with Apple Developer ID and are not notarized yet.
- Windows packages are not code-signed yet.
- Linux runtime support is not available yet.
- Auto-update is not available yet.
- Provider runtime behavior may depend on platform, region and account configuration.
- Local history completeness depends on sync availability and app runtime.
- Diary exports depend on locally available readings.

## Safety disclaimer

GlucoDesk is an awareness companion.

It is not a medical device.

Do not use GlucoDesk for treatment decisions, insulin dosing, emergency alerts, alarms, diagnosis, or as a replacement for approved diabetes applications or medical devices.

For therapy decisions, always use approved medical devices, official medical apps and healthcare professionals.
